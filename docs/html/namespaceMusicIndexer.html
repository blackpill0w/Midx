◆ data_dir

std::string MusicIndexer::data_dir = std::string(getenv("HOME")) + "/.local/share/music-indexer"
inline

The directory where the indexer stores album art and other data. By default it's "~/.local/share/music-indexer", it's better to modify it to be inside your project's data directory (e.g. "~/.local/share/music-player/indexer-data").

Album art is stored in the following manner:

  • The file name is of the form artistName_albumName.
  • If the artist is inexistant, then the file name is of the form __albumName.
